# Quant-Finance-Pairs-Trading-XOM-CVX
A quantitative Pairs Trading strategy leveraging Cointegration (Engle-Granger Test) to exploit mean-reversion in energy stocks ($\text{XOM}$/$\text{CVX}$). Features Grid Search Optimization for robust performance, achieving a Sharpe Ratio of $0.75$ and 6.67% CAGR over 5 years.
